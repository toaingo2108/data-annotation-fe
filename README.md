# data-annotation-fe

### Demo https://data-annotation-fe.vercel.app

### Install

#### 1. Clone project (FE)

```
git clone https://github.com/toaingo2108/data-annotation-fe.git
cd data-annotation-fe
```

#### 2. Install packages

```
yarn
```

or

```
npm install
```

#### 3. Start project

```
yarn dev
```

or

```
npm run dev
```

```
http://localhost:3000
```
